# Amorxs - Student Notes Platform

Amorxs is a simple and easy-to-use platform where students of Class 11 and 12 can **view and upload notes** for all subjects. This project aims to make learning and sharing notes easier for students.

## Features

- View notes for all subjects.
- Upload notes easily for classmates.
- Clean and simple interface.
